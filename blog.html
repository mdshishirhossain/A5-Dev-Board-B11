Question-1: What are the different ways to select an element in the DOM?

There are many different way to select an element in the DOM are getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: It get an element with both tags and text.
innerText: It get an element with just text only.
textContent: It get an element with just text only but it doesn't consider the style. If any element is hidden with display:none the textContent get that element but innerText doesn't get the hidden element text.

Question-4: What is event bubbling in the DOM?

In DOM, when click a child element then it go upward or propagate to its parent element. We can control the propagation.

Question-5: How do you create, add, and remove elements using JavaScript?

I can use document.CreateElement() for create a new element. I can add the created element into another element by using appendChild(). And I can remove element with removeChild().
